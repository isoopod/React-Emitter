# React-Emitter

2D Particle Emitter for React-lua

## Features

- Multithreaded simulation: Particles are simulated in parallel.
- Native rendering: Particles are rendered outside of the React ecosystem for performance.
- Story support: Particles will work inside storybook plugins such as UI Labs.
- Flipbook support.

## Installation

ReactEmitter is only available on [pesde](https://docs.pesde.dev). Add ReactEmitter with the following command:

```sh
pesde add isoopod/react_emitter -a ReactEmitter
```
